# UNP_PROJECT_BANKING_DATA_ANALYSIS_USING_MySQL


I did this project in collaboration with UNP, where I developed a banking database system titled “MERIT_BANK” using MySQL. 
It includes tables like customers, accounts, loans, and branches, and uses SQL queries such as joins and aggregate functions to manage and analyze data efficiently. Additionally, the system reflects real-world banking operations and helps generate insights like customer details, account summaries, and loan analysis, improving my skills in database design and SQL querying.
It also strengthened my understanding of data relationships and normalization. Overall, this project enhanced my ability to work with structured data and perform efficient data analysis.
